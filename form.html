What is a Form?

Forms are used to collect user input. They can include text fields, checkboxes, radio buttons, and more.

Basic Form Structure








Form Elements

  • Text Input: Used for single line text input.
  • Email Input: Used for email addresses.
  • Textarea: Used for multi line text input.
  • Checkboxes: Allow users to select multiple options.
  • Radio Buttons: Allow users to select one option from a set.
